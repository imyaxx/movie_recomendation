# CineMatch

Консольный рекомендатель фильмов на базе AI. Чат задаёт вопросы о твоём настроении, жанровых предпочтениях и последних просмотрах — и подбирает идеальный фильм

---

## Стек

| Компонент   | Технология                                                           |
| ----------- | -------------------------------------------------------------------- |
| Язык        | [Go](https://go.dev) 1.22                                            |
| AI          | [OpenAI API](https://platform.openai.com) — `gpt-4o-mini`            |
| Зависимости | Только стандартная библиотека (`net/http`, `encoding/json`, `bufio`) |
| Интерфейс   | Консоль (терминал)                                                   |

---

## Структура проекта

```
cinematch/
├── main.go         — точка входа, REPL-цикл, выбор языка, UI
├── openai.go       — HTTP-клиент OpenAI API, типы запроса/ответа
├── recommender.go  — системный промпт, состояние сессии, детектор рекомендаций
├── config.go       — все константы (модель, таймауты, маркеры)
├── go.mod          — модуль Go
├── .env.example    — шаблон переменных окружения
└── .gitignore
```

---

## Как это работает

1. При запуске предлагается выбрать язык: **English** или **Русский**
2. AI задаёт 3–5 вопросов о настроении, предпочтениях и последних фильмах
3. На основе ответов подбирает 3 фильма с объяснением, почему именно они подходят
4. Введи `quit` / `выйти` в любой момент для выхода

---

## Конфигурация

Все параметры находятся в [config.go](config.go):

| Константа             | Значение по умолчанию | Описание                          |
| --------------------- | --------------------- | --------------------------------- |
| `DefaultModel`        | `gpt-4o-mini`         | Модель OpenAI                     |
| `DefaultTemperature`  | `0.8`                 | Креативность ответов              |
| `DefaultMaxQuestions` | `5`                   | Максимум вопросов до рекомендации |
| `APITimeout`          | `30s`                 | Таймаут запроса к API             |
